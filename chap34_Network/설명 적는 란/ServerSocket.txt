ServerSocket
	주로 서버측에서 사용 
	
	소캣 통신을 구현할 때 사용되는 클래스
	서버 측에서 클라이언트의 연결을 수락하고
	데이터를 주고받을 수 있게 해줌
	서버를 구현하면 클라이언트가 서버에 연결할 수 있음
	서버는 클라이언트와 통신할 수 있음
	
	
	메서드 							/*서버 포트 번호*/
	ServerSocket sS = new ServerSocket(int 포트번호); 
		지정된 포트 번호로 서버 소캣을 생성
		
	accept() : 클라이언트의 연결을 기다리고, 클라이언트가 연결되면
				클라이언트와 통신할 수 있는 소캣을 반환
				
	
	close() : 서버 소캣을 닫음 
	



	
Socket
	클라이언트 측에서 사용 
	클라이언트는 Socket 객체를 생성해 서버에 연결
	서버 IP주소와 포트번호를 사용해서 특정 서버에 연결하는 역할
	클라이언트와 서버간에 데이터 통신을 가능하게 함 
	
	getInputStream() : 객체에서 입력 스크림을 얻을 때 사용 
	 
	getOutputStream() : 출력 스크림을 얻을 때 사용 
	
	connect() : 클라이언트가 서버에 연결할 때 사용 주어진 서버 주소와 포트 번호로 서버 연결을 시도 
	
	close() : Socket의 객체를 닫을 때 사용
